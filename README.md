# تطبيق متابعة الموظفين

تطبيق لإدارة ومتابعة إجازات الموظفين، المأموريات، والرصيد المرحل من العام السابق والإجازات المرضية.

## الميزات
- عرض قائمة الموظفين
- الاطلاع على تفاصيل كل موظف
- تتبع الإجازات السنوية، المرضية والمأموريات
- حساب وعرض الرصيد السنوي والرصيد المرحل من العام السابق

## التقنيات المستخدمة
- Flutter
- Dart

## تشغيل التطبيق
قم بتنفيذ الأمر التالي في مجلد المشروع:
```bash
flutter run
```
